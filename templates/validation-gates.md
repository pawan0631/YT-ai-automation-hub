# Validation Gates

Use this checklist for every long documentary and related shorts.

## Gate 1: Topic Approval

- [ ] Topic fits channel niche.
- [ ] Strong curiosity gap.
- [ ] High stakes.
- [ ] Visual potential.
- [ ] No obvious compliance blocker.

Decision:

- [ ] Approved
- [ ] Revise
- [ ] Reject

## Gate 2: Research Approval

- [ ] Sources are credible.
- [ ] Timeline is clear.
- [ ] Key claims have references.
- [ ] Weak/uncertain claims are marked.
- [ ] Visual opportunities are identified.

Decision:

- [ ] Approved
- [ ] Revise
- [ ] Reject

## Gate 3: Script Approval

- [ ] Hook is strong.
- [ ] Story has escalation.
- [ ] No unsupported claims.
- [ ] Pacing fits 15-18 minutes.
- [ ] English script approved.
- [ ] Hindi adaptation approved.

Decision:

- [ ] Approved
- [ ] Revise
- [ ] Reject

## Gate 4: Voice Approval

- [ ] English voice fits brand.
- [ ] Hindi voice fits brand.
- [ ] Pronunciation checked.
- [ ] Pace and emotion checked.
- [ ] Audio quality acceptable.

Decision:

- [ ] Approved
- [ ] Revise
- [ ] Reject

## Gate 5: Visual Approval

- [ ] Visual style fits documentary tone.
- [ ] Cinematic quality acceptable.
- [ ] AI-generated assets tracked.
- [ ] Stock/licensed assets tracked.
- [ ] No misleading fake evidence.

Decision:

- [ ] Approved
- [ ] Revise
- [ ] Reject

## Gate 6: Final Upload Approval

- [ ] Final cut reviewed.
- [ ] Copyright/source notes checked.
- [ ] AI/synthetic disclosure checked.
- [ ] Cyber safety checked if applicable.
- [ ] Title approved.
- [ ] Thumbnail approved.
- [ ] Description and chapters approved.
- [ ] Shorts approved.

Decision:

- [ ] Approved for upload
- [ ] Revise
- [ ] Hold
