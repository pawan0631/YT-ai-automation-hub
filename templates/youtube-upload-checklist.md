# YouTube Upload Checklist

## Before Export

- [ ] Script fact-checked.
- [ ] Risky claims reviewed.
- [ ] Music/footage/image rights checked.
- [ ] AI-generated realistic scenes tracked.
- [ ] Audio loudness checked.
- [ ] Captions/subtitles prepared.

## Metadata

- [ ] Title option A.
- [ ] Title option B.
- [ ] Thumbnail option A.
- [ ] Thumbnail option B.
- [ ] Description with sources/credits.
- [ ] Chapters.
- [ ] Tags.
- [ ] Pinned comment.
- [ ] End screen/card plan.

## Shorts Support

- [ ] Short 1 exported.
- [ ] Short 2 exported.
- [ ] Shorts link back to long video.

## After Publish

- [ ] Check first 2 hour CTR.
- [ ] Check first 30 second retention.
- [ ] Respond to early comments.
- [ ] Log lessons in analytics notes.
