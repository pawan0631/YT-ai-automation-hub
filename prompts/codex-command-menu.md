# Codex Command Menu

Use these as command-style requests inside Codex.

## New Episode

```text
Create a new episode plan for:
Channel: Cybersecurity or Business
Topic:
Audience:
Desired tone:

Use the repo pipeline. Produce topic scorecard, research questions, title angles, and validation gates.
```

## Research Dossier

```text
Using the provided sources, create the research dossier for this episode. Keep claims source-backed, mark weak evidence, and suggest visual opportunities.
```

## Script Draft

```text
Using the approved research dossier, draft the English 15-18 minute documentary script with chapter structure and visual direction.
```

## Hindi Version

```text
Adapt the approved English script into natural Hindi. Preserve facts and source-sensitive wording. Include pronunciation notes.
```

## Shorts

```text
Create 2 viral but accurate 1-2 minute shorts in English and Hindi from the approved long script. Include hook, captions, visual direction, and CTA.
```

## Visual Plan

```text
Create a cinematic shot list for the approved script. Separate real footage, stock footage, graphics, AI visuals, maps, and timeline shots.
```

## Upload Package

```text
Create title options, thumbnail concepts, description, chapters, pinned comment, tags, and compliance notes for the approved final video.
```
