# Scripts

Store lightweight helper scripts here.

Suggested scripts:

- `backup-rclone.ps1`: sync project exports and configs to cloud storage.
- `extract-audio.ps1`: extract audio from video using FFmpeg.
- `make-shorts.ps1`: cut approved timestamp ranges into shorts.
- `source-ingest.py`: normalize source notes into a research dossier input format.

Do not put secrets in scripts. Use environment variables or a local `.env` file.
